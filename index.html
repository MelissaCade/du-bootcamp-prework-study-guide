HTML

Concepts

  • The head element contains information about the webpage.
  • Metadata goes in the head element.
  • The body element represents the visible content shown to the user.

HTML Elements

Excellent HTML Elements Reference

  • <!DOCTYPE html> = tells browser what coding language we are using. (goes in the head)
  • <p> = paragraph
  • <h1>,<h2>,etc. = headers in descending size
  • <ul> = unordered list (bullets)
  • <ol> = ordered list (numbered)
  • <dl> = description list (items with descriptions)
    you use <dt> for the list item and <dd> for the description on the following line
  • <li> = list item
  • <u> = underline
  • <br /> = line break
  • <a href = "your link goes here">Link Text</a>
  • <img src="link to image"
    alt="Name/description of image (alt text)"
    width="# of pixels"
    height="# of pixels"
    />

HTML Symbol Codes

HTML Symbol Codes

  • & lt ; = < (less than)
  • & gt ; = > (greater than)
  • & quot ; = " (quotation mark)
  • & # 123 ; = { (opening brace)
  • & # 125 ; = } (closing brace)

CSS

Concepts

  • Inline CSS means changing attributes of one element.
  • You can also set up an internal style sheet in the head that will apply to the whole file.

Attributes

  • <h1 style="color:red;">"TEXT"</h1> = changes the color on one specific header (inline)
  • <head>
    • <style>
    h1 {
    color: red;
    }
    </style>
    </head>

Helpful GIT Commands

  • git status = tells you which branch you're on and whether you're up to date
  • git checkout -b "NAME" = creates new branch called "NAME" (-b means branch)
  • git add -A = tells GIT to stage all files in the current folder
  • git commit -m "COMMIT MESSAGE HERE" = creates a commit with a message
  • git pull origin main = checks whether branch is up to date (helps avoid conflicts)
  • git push origin "NAME" = pushes commit to GITHUB

CLI

Concepts

  • CLI stands for Command-Line Interface.

Useful Commands to Navigate CLI

  • ls = a list of files and directories
  • cd = change directories
  • cd "folder" = drill down to the "folder"
  • cd .. = navigate backwards one directory at a time
  • mkdir = create directory (folder) in currect directory
  • pwd = print working directory (see what folder you're currently in)
  • touch "name-of-file.txt" = create text file called "name-of-file"
  • start "name-of-file.txt" = opens "name-of-file.txt" in text editor
